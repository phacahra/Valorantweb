• Skill Q

    POISON CLOUD ติดตั้งระเบิดประจุควัน กดยิงเพื่อขว้างระเบิดที่จะคงอยู่ตลอดจนจบเกมออกไป กดใช้สกิลอีกครั้งเพื่อสร้างหมอกควันพิษด้วยเชื้อเพลิงที่มีอยู่ สกิลนี้สามารถใช้ซ้ำได้และสามารถหยิบระเบิดกลับขึ้นมาเพื่อใช้งานอีกครั้ง




  • Skill E

    TOXIC SCREEN ติดตั้งเครื่องปล่อยอุปกรณ์ประจุควัน กดยิงเพื่อติดตั้งอุปกรณ์ประจุควันเป็นแนวยาว กดใช้งานอีกครั้งเพื่อสร้างกำแพงควันพิษด้วยเชื้อเพลิงที่มีอยู่ สกิลนี้สามารถใช้ซ้ำได้มากกว่าหนึ่งครั้ง




  • Skill C

    SNAKE BITE ติดตั้งเครื่องยิงสารเคมี ยิงเพื่อปล่อยกระป๋องที่จะระเบิดเมื่อตกถึงพื้น ก่อให้เกิดพื้นที่เขตเคมีที่สร้างดาเมจให้กับศัตรู




  • Skill X

    VIPER’S PIT ติดตั้งเครื่องพ่นควันเคมี กดยิงเพื่อพ่นหมอกควันเคมีออกไปทุกทิศทางรอบ ๆ Viper สร้างกลุ่มหมอกควันขนาดใหญ่ซึ่งจะลดระยะการมองเห็นและค่าพลังชีวิตสูงสุดของผู้เล่นที่ติดอยู่ภายใน

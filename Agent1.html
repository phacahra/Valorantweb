• Skill Q

    INCENDIARY ติดตั้งเครื่องยิงระเบิดเพลิง กดยิงเพื่อปล่อยระเบิดที่จะระเบิดเมื่อตกสู่พื้น ก่อให้เกิดเปลวไฟที่จะสร้างความเสียหายกับผู้เล่นที่อยู่ในบริเวณนั้น




  • Skill E

    SKY SMOKE ติดตั้งแผนที่ยุทธวิธี กดยิงเพื่อกำหนดตำแหน่งที่จะปล่อยหมอกควันของ Brimstone กดโหมดการยิงเสริมเพื่อยืนยันตำแหน่งและปล่อยหมอกควันที่จะบดบังวิสัยทัศน์ในพื้นที่ที่กำหนดไว้เป็นเวลานาน




  • Skill C

    STIM BEACON ติดตั้ง Stim Beacon กดยิงเพื่อวาง Stim Beacon ไปยังเบื้องหน้าของ Brimstone เมื่อวางอยู่บนพื้น Stim Beacon จะเพิ่มประสิทธิภาพในการยิงให้กับผู้เล่นในบริเวณนั้น




  • Skill X

    ORBITAL STRIKE ติดตั้งแผนที่ยุทธวิธี กดยิงเพื่อปล่อยเลเซอร์ Orbital Strike ไปยังพื้นที่ที่กำหนด สร้างดาเมจที่จะเพิ่มความรุนแรงขึ้นเรื่อย ๆ กับผู้เล่นที่อยู่ในพื้นที่นั้น
